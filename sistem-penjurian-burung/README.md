# Sistem Penjurian Burung

**Nama:** Muhammad Faizal

**Mata Kuliah:** Pemrograman Mobile  
**Tujuan:** Pengumpulan Tugas UAS

Project ini adalah sistem penjurian burung yang dibuat untuk memenuhi tugas UAS Pemrograman Mobile. Sistem ini mempermudah proses penilaian burung oleh juri dan menampilkan hasilnya secara otomatis.

## Fitur
- Input nilai juri
- Dashboard peserta dan juri
- Live leaderboard (penilaian real-time)

## Cara Menjalankan
1. Clone repo:
```bash
git clone https://github.com/Rafa-Hits/Pemerograman-Mobile.git
````

2. Jalankan sesuai platform (Flutter):

```bash
flutter pub get
flutter run
```

> Catatan: File `serviceAccountKey.json` bersifat sensitif dan tidak disertakan di repo publik.
